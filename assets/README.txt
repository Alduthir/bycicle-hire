Het beroepsproduct in de bycicle-hire is het
main pakket. Maar maakt wel gebruik van 
items en github.com/gookit/color

Om het beroepsproduct gebruiksklaar te maken:
1. Plaats de bycicle-hire & items mappen in uw GOPATH
2. go get github.com/gookit/color
3. go install items
4. go install van-der-binckes
5. run van-der-binckes.go of voer de executable in de GOBIN uit.
